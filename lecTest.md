# 统计力学如何实际工作：从物理到计算

> **核心思路：一条主线**
> 
> 统计力学解决问题的逻辑链条其实非常清晰：
> 
> $$\text{物理模型} \longrightarrow \text{选系综} \longrightarrow \text{写配分函数} \longrightarrow \text{取对数} \longrightarrow \text{求导得到一切热力学量}$$

**配分函数**是整个框架的枢纽。一旦你算出了配分函数，所有的热力学信息都藏在里面，只需要对不同变量求偏导就能提取出来。这就是为什么统计力学的核心工作几乎总是：**算配分函数**。

---

## 第一步：系综如何选取？

选择系综的原则说白了只有一句话：**哪个量被你的问题固定了，哪个量在涨落，就选对应的系综。**

### 实际判断标准

*   **微正则系综**：极少直接使用。它在概念上最基础（用来推导其他系综），但计算上最痛苦。只有在少数特殊问题中才直接使用，比如计算一个孤立自旋系统的精确微观态数。
*   **正则系综**：最常用。绝大多数实验室条件——样品放在某个温度的环境中——自然对应正则系综。温度是你控制的参数，能量是自由涨落的。固体的比热、磁性材料的磁化率、化学反应的平衡常数……绝大多数教科书习题和实际研究用的都是正则系综。
*   **巨正则系综**：当粒子数不固定或者处理全同粒子统计时使用。典型场景：
    *   量子气体（Bose 气体、Fermi 气体）
    *   吸附问题（气体分子在表面上的吸附，表面上的粒子数在变）
    *   化学平衡（反应物和生成物的粒子数在变）
    *   开放系统（与粒子库接触）

> 💡 **实用经验法则**：先试正则系综，算不动了再换巨正则。

## 第二步：配分函数怎么写？

这是最关键的一步。我用具体例子来展示。

### 例1：二能级系统（最简单的正则系综）

**物理模型**：一个粒子只有两个能级，基态能量为 $0$，激发态能量为 $\varepsilon$。系统与温度为 $T$ 的热库接触。

**写配分函数**：正则系综的配分函数是对所有微观态求和 $e^{-\beta E_i}$（其中 $\beta = 1/k_BT$）：

$$Z = e^{-\beta \cdot 0} + e^{-\beta \varepsilon} = 1 + e^{-\beta \varepsilon}$$

就这么简单。两个态，两项求和，结束。

**提取热力学量**：

*   **自由能**：
    $$F = -k_BT \ln Z = -k_BT \ln(1 + e^{-\beta\varepsilon})$$

*   **平均能量**（对 $\beta$ 求导）：
    $$\langle E \rangle = -\frac{\partial \ln Z}{\partial \beta} = \frac{\varepsilon}{e^{\beta\varepsilon} + 1}$$

*   **比热**（对 $T$ 求导）：
    $$C = \frac{\partial \langle E \rangle}{\partial T} = k_B (\beta\varepsilon)^2 \frac{e^{\beta\varepsilon}}{(e^{\beta\varepsilon}+1)^2}$$

**注意这个比热的行为**：
- 低温 $T \to 0$：$C \to 0$（系统冻在基态，没有热激发）
- 高温 $T \to \infty$：$C \to 0$（两个态等概率占据，再加温也不改变分布）
- 中间某个温度有一个峰（**Schottky 峰**）

> 这个峰的物理意义很直观：在 $k_BT \sim \varepsilon$ 时，热涨落恰好能激发粒子在两个能级之间跳跃，系统对温度变化最“敏感”。

### 例2：理想气体（正则系综的经典应用）

**物理模型**：$N$ 个无相互作用的经典粒子在体积 $V$ 的盒子里，温度 $T$。

**写配分函数**：每个粒子独立，所以 $N$ 粒子配分函数是单粒子配分函数的乘积（除以 $N!$ 是因为全同粒子不可区分）：

$$Z_N = \frac{z^N}{N!}$$

单粒子配分函数 $z$ 是对单个粒子的所有状态求和。经典情况下，“求和”变成相空间积分：

$$z = \frac{1}{h^3}\int d^3r \int d^3p \, e^{-\beta p^2/2m}$$

位置积分给出体积 $V$；动量积分是高斯积分：

$$z = \frac{V}{h^3} \left(\int_{-\infty}^{\infty} dp \, e^{-\beta p^2/2m}\right)^3 = \frac{V}{h^3}(2\pi mk_BT)^{3/2} = \frac{V}{\lambda^3}$$

这里 $\lambda = h/\sqrt{2\pi mk_BT}$ 是**热德布罗意波长**——粒子在温度 $T$ 下的“量子模糊尺度”。它给出了一个极其直觉的图像：$z = V/\lambda^3$ 就是“盒子里能装多少个热波包”。

**提取热力学量**：

$$F = -k_BT \ln Z_N = -k_BT\left[N\ln\frac{V}{\lambda^3} - \ln N!\right]$$

用 Stirling 近似 $\ln N! \approx N\ln N - N$：

$$F = -Nk_BT\left[\ln\frac{V}{N\lambda^3} + 1\right]$$

*   **压强**（对 $V$ 求导）：
    $$P = -\frac{\partial F}{\partial V}\bigg|_{T,N} = \frac{Nk_BT}{V}$$
    这就是 $PV = Nk_BT$——理想气体状态方程，从统计力学中自然地流出来了。

*   **能量**：
    $$\langle E \rangle = -\frac{\partial \ln Z_N}{\partial \beta} = \frac{3}{2}Nk_BT$$
    **能量均分定理**：每个自由度贡献 $\frac{1}{2}k_BT$，三维空间三个平动自由度，所以 $\frac{3}{2}k_BT$。

### 例3：表面吸附（巨正则系综的典型应用）

这是一个用正则系综很难算、但用巨正则系综极其优雅的问题。

**物理模型**：一个固体表面有 $M$ 个吸附位点。气体分子可以从气相吸附到表面，也可以脱附回气相。每个位点要么空着，要么被一个分子占据（占据时能量降低 $\varepsilon$）。表面与气相构成粒子库，化学势 $\mu$ 由气相决定。

**为什么选巨正则？**
因为表面上的粒子数 $N$ 不固定——分子不断地吸附和脱附。如果用正则系综，你需要对固定 $N$ 算配分函数，然后再考虑 $N$ 的变化，这非常繁琐。

**写巨配分函数**：关键的简化在于——**每个吸附位点是独立的！** 所以我们只需要算单个位点的巨配分函数。

单个位点只有两种状态：
- **空**：$N=0$，$E=0$，贡献 $e^{-\beta(0 - \mu \cdot 0)} = 1$
- **占据**：$N=1$，$E=-\varepsilon$，贡献 $e^{-\beta(-\varepsilon - \mu \cdot 1)} = e^{\beta(\varepsilon + \mu)}$

$$\xi = 1 + e^{\beta(\varepsilon + \mu)}$$

$M$ 个独立位点的总巨配分函数：

$$\Xi = \xi^M = \left(1 + e^{\beta(\varepsilon+\mu)}\right)^M$$

**平均占据率**（每个位点被占据的概率）：

$$\langle n \rangle = \frac{e^{\beta(\varepsilon+\mu)}}{1 + e^{\beta(\varepsilon+\mu)}} = \frac{1}{e^{-\beta(\varepsilon+\mu)} + 1}$$

定义 $\theta = \langle n \rangle$ 为覆盖率，利用理想气体的化学势 $\mu = k_BT \ln(P/P_0)$（$P$ 是气相压强），就得到：

$$\theta = \frac{P/P_1}{1 + P/P_1}$$

其中 $P_1$ 是一个与温度有关的特征压强。这就是著名的 **Langmuir 吸附等温线**。
- 低压时 $\theta \propto P$（线性吸附）
- 高压时 $\theta \to 1$（表面饱和）

> 这就是巨正则系综的威力：粒子数的涨落被自然地编码在了数学框架中，你不需要额外处理。

## 直觉总结：三种系综的操作手册

| 步骤 | 微正则 (Microcanonical) | 正则 (Canonical) | 巨正则 (Grand Canonical) |
| :--- | :--- | :--- | :--- |
| **输入约束** | 能级结构 + 固定 $E$ | 能级结构 + 温度 $T$ | 能级结构 + $T$ + 化学势 $\mu$ |
| **核心计算** | 数微观态 $\Omega(E)$ | 算 $Z = \sum_i e^{-\beta E_i}$ | 算 $\Xi = \sum_{N,i} e^{-\beta(E_i - \mu N)}$ |
| **取对数** | 熵 $S = k_B\ln\Omega$ | 自由能 $F = -k_BT\ln Z$ | 巨势 $\Phi = -k_BT\ln\Xi$ |
| **求导得热力学量** | $1/T = \partial S/\partial E$ <br> $P/T = \partial S/\partial V$ | $\langle E\rangle = -\partial_\beta \ln Z$ <br> $P = -\partial_V F$ <br> $S = -\partial_T F$ | $\langle N\rangle = k_BT\,\partial_\mu \ln\Xi$ <br> $P = -\partial_V \Phi$ <br> $S = -\partial_T \Phi$ |

你可以把这个过程想象成一台机器：

```mermaid

    B --> C(取对数)
    C --> D(求偏导)
    D --> E[一切热力学量]

```

> **所有的物理都在第一步**（建立微观模型、写出能级），**所有的数学都在中间**（算配分函数），**所有的热力学都在最后**（求导）。一旦你掌握了这条流水线，统计力学的大部分问题都是在不同的物理模型上重复这个流程。

## 💡 选择系综的思维实验

假设你要研究以下四个问题，看你的系统与外界交换什么，就释放对应的约束：

1. **一块铁的磁性随温度的变化**
   * **情境**：铁块放在炉子里加热。粒子数固定（铁原子不会跑掉），温度由炉子控制。
   * **选择**：**正则系综**，写 Ising 模型的配分函数。
2. **氢气在铂表面的催化吸附**
   * **情境**：氢分子从气相吸附到铂表面，粒子数在变化，温度和气相压强（化学势）由外部控制。
   * **选择**：**巨正则系综**，如上面的 Langmuir 例子。
3. **宇宙微波背景辐射的光子气体**
   * **情境**：光子数不守恒（随时在产生和湮灭），$\mu = 0$（光子的化学势为零，因为光子数不是守恒量）。
   * **选择**：**巨正则系综**，直接推出 Planck 黑体辐射公式。
4. **一个孤立的小原子核的能级密度**
   * **情境**：能量精确已知，没有热库。
   * **选择**：**微正则系综**，计算给定能量下的量子态数。
